PowerShell Web Access és web service

Nagyon érdekes távoli végrehajtási lehetőséget jelent a Windows Server 8-on várható PowerShell Web Access, mellyel böngészőn keresztül tudunk távoli PowerShell utasításokat végrehajtani. Másik érdekes lehetőség, hogy PowerShell szkriptek web szolgáltatásként is futtathatók, és ezek eredménye ugyanolyan objektumként kapjuk meg, mint ahogyan általában a webszolgáltatásoknál tapasztaltuk.
